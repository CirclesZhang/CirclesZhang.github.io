Model Context Protocol Server 初体验

背景

模型上下文协议（MCP）是由AI安全和研究公司Anthropic开发的一个开放标准，旨在解决AI助手与外部数据源和工具集成的问题。它于2024年11月25日推出，提供了标准化连接的方式，使LLM（大型语言模型）能够更轻松地访问和处理外部数据。MCP被比喻为AI应用的USB-C端口，简化了构建代理和复杂工作流的过程。

目标

本文档目标于Windows环境下使用Python、Java 代码实现简单的 MCP server【天气的查询】，并测试实现至Java MCP client 和 Cherry Studio。

准备

  • Python环境
    • Python 3.10
    • Python MCP SDK 1.2.0
  • Java环境
